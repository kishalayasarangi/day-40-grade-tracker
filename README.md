# Day 40 — Student Grade Tracker

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A student grade tracker that calculates GPA, CGPA,
subject-wise performance and generates a report card.

## How to view
Open `index.html` in your browser.

## Features
- Add subjects with credits and marks
- Auto calculates grade points and GPA
- CGPA tracker across semesters
- Performance chart per subject
- Export report card as text
